# cra-redux-toolkit

Redux Toolkit の雛形

- React 17.0.2
- Redux-Toolkit: 1.8.0
- TypeScript 4.1.5
- react-router-dom 6.2.2
- axios 0.26.0
- normalizr: 3.6.1
- MUI 5.4.4
- Storybook 6.4.19

# Usage

1. Start json-server.

```bash
$ npm run mock
```

2. Launch the React app.

```bash
$ npm start
```

3. Check the Storybook.

```bash
$ npm run storybook
```
